# Esri grid 
Esri grid formaatteja on kahdenlaisia, binääri- ja tekstimuotoisia. Binäärimuotoisista käytetään yleisesti nimitystä ARC GRID.

**Mille ohjelmalle kehitetty?** 
ArcGIS

**Binäärimuoto, esim. ARC Grid**
Käytetään yleensä ohjelman sisäisesti.

**Tekstimuoto, esim. ASCII Grid**
Käytetään tiedon jakamiseen eli ns. siirtoformaatti.

Kuka kehittänyt?
- ESRI
